Cómo subir tus modelos .glb gratis
⭐ Opción 1 — GitHub (recomendado)
Sube el .glb a un repositorio público → copia la URL Raw. Sin problemas de CORS.
Opción 2 — Carpeta local
Pon el archivo en assets/models/ y escribe la ruta: assets/models/mi-pieza.glb
⚠ Google Drive
Pega el link de compartir y lo convertiremos automáticamente. Puede fallar por CORS si Google cambia sus políticas.

Formatos: solo .glb / .gltf funcionan en el visor web. Convierte STL/OBJ/3MF con Blender (gratis) o Aspose online.

Agregar filamento

Selecciona un preset para ver los valores recomendados. Puedes ajustar con los sliders después.

0 = espejo/vidrio · 0.3 = seda brillante · 0.8-1.0 = mate opaco

0 = plástico/PLA normal · 0.85-1.0 = acabado cromado real. Seda/silk deja en 0 — su brillo viene del roughness bajo.

1.0 = sólido · 0.3 = translúcido (PETG transparente)

Si lo dejas vacío, se muestra la esfera por defecto. Sube un .glb para mostrar tu propio modelo al seleccionar este filamento.
